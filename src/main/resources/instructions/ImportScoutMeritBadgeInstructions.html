Scout Merit Badge Import

Files that are imported must have a .csv extension. Make sure that all your values are separated by commas.
This import is used to link existing scouts to existing merit badges. If the values in the import do not exists in BSA Database the import will fail.
The following table will explain what data is valid and how to build our import file. The type and max values are related to the type of information that will be stored in the associated field (such as: Text or Number) and the maximum number of character allow in each field respectfully.

Scout Name (Required)
(type, max)		 The name of the scout you are linking to a merit badge.
(Text, 254)
Scout Type Name (Required)
(type, max)		 Scout Type Name, accepted values are: Cub Scout, Boy Scout, Varsity Scout, or Venture Scout.
(Text, N/A)
Merit Badge Name (Required)
(type, max)		 The name of the merit badge you are linking to the scout.
(Text, 254)

Example

"Scout Name","Scout Type Name","Merit Badge Name"
"Steve Rogers","Boy Scout","Camping"
"Steve Rogers","Boy Scout","Swimming"
"Steve Rogers","Boy Scout","Cooking"
