Boy Scout Import

Files that are imported must have a .csv extension. Make sure that all your values are separated by commas.
It is important to note that when you specify a boy scout name that already exists in BSA Database this will update the existing data.
The following table will explain what data is valid and how to build the import file. The type and max values are related to the type of information that will be stored in the associated field (such as: Text or Number) and the maximum number of character allow in each field respectfully.

Boy Scout Name (Required)
(type, max)		 The name of the boy scout.
(Text, 90)
Birth Date (Required)
(type, max)		 Boy scout's birth date. Please format date as "YYYY-MM-DD".
(Text, N/A)
Advancement (Required)
(type, max)		 Name of boy scout's current rank advancement. (the advancement needs to already exists in BSA Database)
(Text, 254)
Advancement Date (Required)
(type, max)		 The date the boy scout's current rank advancement was received. Please format date as "YYYY-MM-DD".
(Text, N/A)
Position (Optional)
(type, max)		 The boy scout's current position in the troop.
(Text, 254)
Scout Phone Number (Optional)

(type, max)		 The boy scout's phone number.
(10 digit number formatted like a phone number)
(Text, N/A)
City (Optional)
(type, max)		 Boy scout's address info.
(Text, 254)
State (Optional)
(type, max)		 Boy scout's address info.
(Text, 254)
Street (Optional)
(type, max)		 Boy scout's address info.
(Text, 254)
Zip (Optional)
(type, max)		 Boy scout's address info.
(Text, 254)
Parent Name (Optional)
(type, max)		 Name of scout's primary parent or guardian.
(Text, 254)
Parent Phone Number (Optional)

(type, max)		 The boy scout's primary parent or guardians phone number.
(10 digit number formatted like a phone number)
(Text, N/A)
Notes (Optional)
(type, max)		 Notes on the boy scout.
(Text, N/A)

Example

"Boy Scout Name","Birth Date","Advancement","Advancement Date","Position","Scout Phone Number","City","State","Street","Zip","Parent Name","Parent Phone Number","Notes"
