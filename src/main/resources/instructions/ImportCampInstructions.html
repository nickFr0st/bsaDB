Camp Import

Files that are imported must have a .csv extension as such make sure that all your values are separated by commas.
It is important to note that when you specify a camp name that already exists in BSA Database this will update the existing data. To keep existing scouts attended you must specify them in the import.
When adding scouts attended please list all scout names of attending scouts after the camp. To signify a new camp leave an empty space between the last scout attended and the name of the next camp.
The following table will explain what data is valid and how to build our import file. The type and max values are related to the type of information that will be stored in the associated field (such as: Text or Number) and the maximum number of character allow in each field respectfully.

Camp Name (Required)
(type, max)		 The name of the camp.
(Text, 254)
ScoutTypeName (Required)
(type, max)		 Scout Type Name, accepted values are: Cub Scout, Boy Scout, Varsity Scout, or Venture Scout.
(Text, N/A)
Location (Required)
(type, max)		 The Location of the camp.
(Text, 254)
Start Date (Required)
(type, max)		 The date the camp started on. Please format date as "YYYY-MM-DD".
(Text, 9)
Leaders (Required)
(type, max)		 The leaders that attended the camp. format value as "leader name1, leader name2, ..."
(Text, 254)
Notes (Optional)
(type, max)		 The camp notes.
(Text, N/A)
Scouts Attended (Optional)
(type, max)		 Add a scout name per line for each scout that has attended this camp.
(Text, N/A)

Example

"Camp Name","Scout Type Name","Location","Start Date","Leaders","Notes"
"Scouts Attended"
"Maple Dell","Boy Scout","Payson Canyon","2014/08/01","Leader1, Leader2","The boys had a lot of fun, we should do this again."
"Jimmy Tom"
"Luke Dale"
"Dave Thomas"
