Service Project Import

Files that are imported must have a .csv extension as such make sure that all your values are separated by commas.
It is important to note that when you specify a service project name that already exists in BSA Database this will update the existing data. When updating a service project to keep existing scouts attended you must specify them in the import as well.
When adding scouts attended please list all scout names of attending scouts after the service project. To signify a new service project leave an empty space between the last scout attended and the name of the next service project.
The following table will explain what data is valid and how to build the import file. The type and max values are related to the type of information that will be stored in the associated field (such as: Text or Number) and the maximum number of character allow in each field respectfully.

Service Project Name (Required)
(type, max)		 The name of the service project.
(Text, 254)
ScoutTypeName (Required)
(type, max)		 Scout Type Name, accepted values are: Cub Scout, Boy Scout, Varsity Scout, or Venture Scout.
(Text, N/A)
Location (Required)
(type, max)		 The Location of the service project.
(Text, 254)
Start Date (Required)
(type, max)		 The date the service project started on. Please format date as "YYYY-MM-DD".
(Text, 9)
Leaders (Required)
(type, max)		 The leaders that attended the service project. format value as "leader name1, leader name2, ..."
(Text, 254)
Duration (Required)

(type, max)		 Time spent at service project."
(Must be a positive number)
(Number, N/A)
Notes (Optional)
(type, max)		 The service project notes.
(Text, N/A)
Scouts Attended (Optional)
(type, max)		 Add a scout name per line for each scout that has attended this camp.
(Text, N/A)

Example

"Service Project Name","Scout Type Name","Location","Start Date","Leaders","Duration","Notes"
"Scouts Attended"
"Fall Leaf Raking","Boy Scout","Santaquin Area","2016/08/01","Leader1, Leader2","2.5","We raked the whole area and it was awesome!."
"Jimmy Tom"
"Luke Dale"
"Dave Thomas"
