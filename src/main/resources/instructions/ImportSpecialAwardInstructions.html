Special Award Import

Files that are imported must have a .csv extension. Make sure that all your values are separated by commas.
This import is used to add special awards to existing scouts. If the specified scouts do not exists in BSA Database the import will fail.
The following table will explain what data is valid and how to build our import file. The type and max values are related to the type of information that will be stored in the associated field (such as: Text or Number) and the maximum number of character allow in each field respectfully.

Scout Name (Required)
(type, max)		 The name of the scout you are adding the special award to.
(Text, 254)
Scout Type Name (Required)
(type, max)		 Scout Type Name, accepted values are: Cub Scout, Boy Scout, Varsity Scout, or Venture Scout.
(Text, N/A)
Award Name (Required)
(type, max)		 The name of the special award you are adding to the scout.
(Text, 254)
Award Description (Required)
(type, max)		 The special awards' description.
(Text, 254)
Award Date Received (Required)
(type, max)		 The date the scout received this special award. Please format date as "YYYY-MM-DD".
(Text, N/A)

Example

"Scout Name","Scout Type Name","Award Name","Award Description","Award Date Received"
"Steve Rogers","Boy Scout","Frontiersman Award","Learned the way of the frontiersman","2016/08/05"
"Steve Rogers","Boy Scout","Liahona Award","Decisions Determine Destiny","2016/08/04"
